# Testapp Projesi

Basit bir değerlendirme projesidir.

## Getting Started

Ruby on Rails ile oluşturulmuş bu projeyi, bilgisayarına kopyalamanı ve sana gönderdiğimiz mail doğrultusunda, gerekli geliştirmeleri yapmanı istiyoruz.

### Installing

Bu projeyi ayağa kaldırmak için gerekli kurulumlara aşağıdaki linkten ulaşabilirsin. İşletim sisteminin versiyonuna dikkat etmen gerek. Ayrıca "rbenv" ve "postgresql" secenekleri üzerinden kurulumlarını gerçekleştirmeni istiyoruz.

https://gorails.com/setup/osx/10.14-mojave

### Final Step
Projeyi, kendi github hasabında oluşturacağın, yeni bir repository e push etmeni ve tamamladığında bize mail ile bu repository nin link ini atmanı istiyoruz.

Başarılar.
